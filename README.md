# 编程工具的集合
## 一、目录结构说明
### 1、debug 目录
* debug目录为显示调试信息的接口集合，实现信息输出的相关接口。
### 2、file 目录
* file目录为文件操作的相关接口集合，实现关于文件的相关接口。
### 3、list 目录
* list目录为链表操作的相关接口集合，实现关于链表的相关接口。
### 4、share_mem 目录
* share_mem目录为共享内存接口集合，对system V共享内存接口的外层封装，方便使用。
### 5、semaphore 目录
* semaphore目录为信号量接口集合，对system V信号量接口的外层封装，方便使用。
